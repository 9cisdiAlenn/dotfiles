## info and requirements

- WM: bspwm
- notification system: dunst
- shell: zsh 
- zsh theme: common
- OS: arch linux
- keybinding handler: sxhkd
- editor: neovim (with nvchad)
- bar: polybar
- terminal: tym and kitty
- app launcher: rofi
- audio visualizer: cava
- compositor: picom (not configurated)
